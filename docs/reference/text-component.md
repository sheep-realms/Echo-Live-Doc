# 本地化文本组件

**本地化文本组件**是一种用于注册表的某些文本属性中的数据结构。

## 包含字段

### `text`

默认文本内容。如果本地化键名未命中，则回退输出此字段中的内容。如果未定义此字段，则以本地化键名原样输出。

### `translate`

本地化键名。

通常情况下，特定的功能模块会事先提供一个可复用的本地化键名前缀，只需填写相对路径，从而节省不必要的路径。

从 1.7.8 版本开始，可以使用冒号 `:` 前缀来忽略默认前缀，使用绝对路径。

### `with`

本地化变量，类型为 Object。

## 示例

=== "本地化文本组件"

    ``` json
    {
        "text": "示例项目：（空）",
        "translate": "example.foo.bar",
        "with": {
            "name": "项目名称"
        }
    }
    ```

=== "本地化文件"

    ``` json
    {
        "example": {
            "foo": {
                "bar": "示例项目：{name}"
            }
        }
    }
    ```
