# 本地化文本组件

本地化文本组件是一种用于注册表的某些文本属性中的数据结构。

## 包含字段

### `text`

默认文本内容。如果本地化键名未命中，则回退输出此字段中的内容。如果未定义此字段，则以本地化键名原样输出。

### `translate`

本地化键名。

### `with`

本地化变量，类型为 Object。

## 示例

=== "本地化文本组件"

    ``` json
    {
        "text": "示例项目：（空）",
        "translate": "example.foo.bar",
        "with": {
            "name": "项目名称"
        }
    }
    ```

=== "本地化文件"

    ``` json
    {
        "example": {
            "foo": {
                "bar": "示例项目：{name}"
            }
        }
    }
    ```
