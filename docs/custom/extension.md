# 扩展

!!! info "扩展功能已初步实现"

    截至 1.8.0 版本，扩展系统已迭代到 V2，但功能尚未完善，您可以通过编辑根目录中的 `extensions.js` 来选择载入哪些扩展。`extensions` 文件夹中已有一个示例扩展以供参考。

Echo-Live 支持扩展，可用于增加额外资源。

**注意：Echo-Live 的扩展仅作规范导入分享资源之用，实际上可以被用来做任何事，因此请不要安装来路不明的扩展。**

## :material-import: 安装

1. 将扩展（是一个文件夹）放入 `extensions` 文件夹。
2. 在 `extensions.js` 中插入该扩展的文件夹名称。
